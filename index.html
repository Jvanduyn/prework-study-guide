Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The general template for HTML is known as a boilerplate
  • You can pull up the boiler plate with the ! command
  • ul and ol are used for lists. Either with numbers or non numbered

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be chosen on the HEX rgb scale
  • you can move different things in CSS by using the PX which means Pixels
  • the command "box-shadow" will have a shadow casted however many pixels around the text box assigned

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
