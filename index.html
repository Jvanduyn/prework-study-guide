HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The general template for HTML is known as a boilerplate
  • You can pull up the boiler plate with the ! command
  • ul and ol are used for lists. Either with numbers or non numbered

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be chosen on the HEX rgb scale
  • you can move different things in CSS by using the PX which means Pixels
  • the command "box-shadow" will have a shadow casted however many pixels around the text box assigned

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A will stage the changes
  • git pull origin main will make sure our local branch is up to date
  • git push will push the changes made to git hub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • You need to call a function after listing it
  • math random is used to call a random number which can be used to get a random if else statement
  • If else statements are used when a response is needed after triggering something
